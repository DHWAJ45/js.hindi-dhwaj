# EpicBliss Solutions Website

A professional business website for EpicBliss Solutions, showcasing their IT services including SharePoint, Power Platform, Business Intelligence, RPA, and more.

## Overview

This is a modern, responsive website built for EpicBliss Solutions, a technology consulting company with 16+ years of experience. The website features:

- **Services Showcase**: SharePoint, Power BI, RPA, Salesforce, AWS, Mobile Development
- **Interactive Portfolio**: Live BI reports and brochures
- **Company Information**: About section, client testimonials, statistics
- **Contact Forms**: Multiple ways to get in touch
- **Popup Marketing**: Hire developers popup with smooth animations

## Features

### 🎨 Design & UX
- Responsive design that works on all devices
- Modern gradient backgrounds and smooth animations
- Professional color scheme with blue and green accents
- Owl Carousel for service and portfolio sections
- Custom popup modal with engaging call-to-action

### 🛠️ Technical Stack
- **HTML5** with semantic markup
- **CSS3** with modern features (gradients, transitions, flexbox)
- **Bootstrap 5.3.2** for responsive grid system
- **jQuery 3.5.1** for DOM manipulation
- **Owl Carousel 2.3.4** for interactive sliders
- **Font Awesome 6.4.0** for icons
- **Google Fonts** (Poppins, Ubuntu, Roboto Slab)

### 📱 Responsive Features
- Mobile-first design approach
- Collapsible navigation menu
- Optimized popup for smaller screens
- Touch-friendly carousel controls
- Responsive image sizing

### 🎯 Business Sections
1. **Hero Banner**: Rotating slides showcasing main services
2. **About Us**: Company overview with 16 years experience
3. **Services**: 9 core service offerings with descriptions
4. **Statistics**: Client metrics (150+ clients, 100+ projects)
5. **Portfolio**: Downloadable brochures and case studies
6. **Live BI Reports**: Interactive Power BI and Tableau dashboards
7. **Contact**: Multiple contact methods and locations

## Required Assets

To run this website properly, you'll need to create the following directory structure and add the missing files:

### CSS Files (assets/css/)
- `fontawesome.css` - Font Awesome icon styles
- `templatemo-finance-business.css` - Main theme styles
- `owl.css` - Owl Carousel custom styles

### JavaScript Files (assets/js/)
- `custom.js` - Custom website functionality
- `owl.js` - Owl Carousel initialization
- `slick.js` - Slider functionality
- `accordions.js` - Accordion components

### Images (assets/images/)
- `image.png` - Developer icon for popup
- `Epicbliss Logo.png` - Company logo
- `Banner 1 image.png` - About section image
- `SliderImages/` - Service icons (SharePoint, BI, RPA, etc.)
- `BrochureImages/` - Portfolio thumbnails
- `BI Portfolio/` - BI report preview images

### Vendor Files (vendor/)
- `bootstrap/css/bootstrap.min.css`
- `bootstrap/js/bootstrap.bundle.min.js`
- `jquery/jquery.min.js`

## Setup Instructions

1. **Clone or download** this project
2. **Create the assets directory structure** as shown above
3. **Add the missing CSS, JS, and image files**
4. **Open index.html** in a web browser
